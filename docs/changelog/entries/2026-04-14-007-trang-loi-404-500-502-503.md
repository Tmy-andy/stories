## Mục đích

Giải quyết dòng số 6 (bổ sung) trong `Note để lần sau sửa.md`: thiết kế các trang lỗi 404, 500, 502, 503 để trải nghiệm lỗi không còn là màn hình trắng.

## Thiết kế

- 1 component shared `ErrorPage.jsx` — nhận `code`, `title`, `message`, `icon` → render layout: code lớn, icon, tiêu đề, mô tả, CTA (Về trang chủ / Quay lại).
- 4 preset: `NotFound` (404), `ServerError` (500), `BadGateway` (502), `ServiceUnavailable` (503).
- `ErrorBoundary` class component bọc toàn app → bắt crash runtime → hiện `ServerError`.
- Catch-all `*` route trong `/*` layout → `NotFound` (giữ Header/Footer).
